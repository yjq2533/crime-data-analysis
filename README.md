# Crime Data Analysis (R)

# Overview
This project analyzes crime data using R, focusing on loops, data manipulation, and visualization.

# Key Features
- Implemented manual counting using nested for-loops
- Analyzed crime trends over time
- Built reusable functions for crime analysis

# Tools
- R
- tidyverse (dplyr, ggplot2)
- lubridate

# Data
The dataset (crime2.csv) is provided as part of course materials and is not included in this repository.
